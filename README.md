# Microsoft-Excel-Project# Sales Performance & Profitability Analysis Dashboard

## Overview

This project is an interactive Excel dashboard created using the Financial Sample dataset. It analyzes sales performance, profitability, customer segments, products, and country-wise trends.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers

## Key Insights

* Government segment generated the highest sales and profit.
* Channel Partners achieved the highest profit margin.
* Enterprise segment recorded a loss.
* Sales showed significant growth from 2013 to 2014.
